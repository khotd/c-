# Лабораторная работа 3. Реализация коллекций в C#
Суть работы: реализовать собственные версии базовых коллекций в .NET, изучить их внутреннюю структуру, интерфейсы и принципы работы.
Выполнила: Хотеева Диана К0709-23/3

# Классы и структура проекта
# Основные коллекции:
1. SimpleList - собственная реализация динамического массива с интерфейсами IList<T>, ICollection<T>, IEnumerable<T>

2. SimpleDictionary - собственная реализация хеш-таблицы с цепочками с интерфейсом IDictionary<TKey, TValue>

3. DoublyLinkedList - собственная реализация двусвязного списка с интерфейсом IList<T>

# Тесты:
1. SimpleListTests - модульные тесты для проверки корректности операций SimpleList

2. SimpleDictionaryTests - модульные тесты для проверки корректности операций SimpleDictionary

3. DoublyLinkedListTests - модульные тесты для проверки корректности операций DoublyLinkedList

# Основные технологии
1. .NET 8.0 - платформа разработки

2. C# 11 - язык программирования

3. xUnit 2.6.3 - фреймворк для unit-тестирования

# Используемые библиотеки
1. System.Collections - базовые интерфейсы коллекций

2. System.Collections.Generic - обобщенные коллекции

3. System - базовые типы и исключения

# Тестирование:
1. xUnit 2.7.0 - фреймворк для unit-тестирования

2. Microsoft.NET.Test.Sdk 17.9.0 - SDK для тестирования
