# Лабораторная работа 4. Синхронизация потоков в C#
Суть работы: Изучить проблему синхронизации потоков в многопоточных приложениях, научиться выявлять и предотвращать взаимоблокировки (deadlock), реализовать классические задачи синхронизации.
Выполнила: Хотеева Диана К0709-23/3

# Классы и структура проекта:
Модели для задачи "Обедающие философы" (Dining Philosophers):
1. PhilosopherWithDeadlock.cs - реализация с гарантированным deadlock (демонстрация проблемы)

2. PhilosopherFixed.cs - исправленная версия без deadlock

Модель для задачи "Спящий парикмахер" (Sleeping Barber):
1. BarberShop.cs - реализация парикмахерской с ограниченной очередью клиентов

Модели для задачи "Производитель-Потребитель" (Producer-Consumer):
1. ProducerConsumerBlockingCollection.cs - реализация с использованием BlockingCollection<T>

2. ProducerConsumerManual.cs - реализация с использованием SemaphoreSlim + lock


# Основные технологии:
1. .NET 8.0 - платформа разработки

2. C# 11 - язык программирования

3. Многопоточность и синхронизация
